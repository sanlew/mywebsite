# mywebsite

It's simple one-page layout using Angular 6.

Run local:
1. `git clone https://github.com/sanlew/mywebsite.git`
2. `npm install`
3. `ng serve`
4. Navigate to `http://localhost:4200/`.


Preview online: `https://sanlew.github.io/mywebsite/`
